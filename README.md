# 🌊 Ocean Personal Blog - Zaskia Qanita

Proyek ini adalah blog pribadi interaktif dengan tema estetika laut  Dikembangkan sebagai bagian dari tugas pemrograman web untuk menampilkan identitas mahasiswa, daftar keahlian, serta implementasi interaksi dinamis menggunakan JavaScript.

## 🚀 Fitur Utama
* Desain Responsif**: Tampilan yang menyesuaikan dengan perangkat mobile maupun desktop.
* Interaksi DOM**: Manipulasi elemen HTML seperti perubahan teks judul secara real-time melalui JavaScript.
* Modal Kontak**: Pop-up interaktif (pop-up lucu) untuk memudahkan pengunjung menghubungi pemilik blog.
* Formulir Identitas**: Validasi input sisi klien untuk memastikan data nama, email, dan alamat terisi sebelum dikirim.
* Tema Estetik**: Palet warna biru laut (*ocean dark*, *mid*, *light*) yang bersih dan profesional.

## 🛠️ Teknologi yang Digunakan
* HTML5: Digunakan untuk struktur dasar dan elemen semantik halaman.
* CSS3: Implementasi variabel warna (`:root`), tata letak, dan efek glassmorphism pada navbar.
* JavaScript (ES6+): Digunakan untuk logika interaksi, penggunaan Arrow Functions, dan manajemen event handling.
