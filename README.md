# 🌊 Portofolio Digital & Blog Pribadi: Ocean Aesthetic

Selamat datang di repositori blog pribadi **Zaskia Qanita Najiyah**. Proyek ini bukan sekadar tugas web, melainkan representasi digital yang menggabungkan estetika visual bertema laut dengan fungsionalitas pemrograman web modern.

## 📖 Deskripsi Proyek

Web ini dirancang dengan konsep **"Oceanic Clarity"**, menggunakan palet warna yang terinspirasi dari kedalaman samudera (Deep Sea Blue) dan kehangatan terumbu karang (Coral Orange). Fokus utama dari proyek ini adalah menciptakan pengalaman pengguna yang bersih, responsif, dan interaktif bagi pengunjung yang ingin mengenal lebih jauh tentang profil akademik dan keahlian teknis saya.

## ✨ Keunggulan Visual & Fitur

* **Antarmuka "Ocean Theme"**: Menggunakan sistem variabel CSS untuk menciptakan skema warna yang konsisten antara biru gelap, biru muda, dan aksen oranye yang kontras.
* **Profil Akademik Dinamis**: Menampilkan identitas sebagai mahasiswa Teknik Informatika Universitas Muhammadiyah Sukabumi (UMMI) dengan elemen visual yang profesional.
* **Sistem Greeting Berbasis Waktu**: JavaScript secara cerdas mendeteksi waktu lokal pengunjung untuk memberikan salam yang personal (Pagi/Siang/Malam) pada bagian Hero.
* **Manajemen Data Terintegrasi**: Dilengkapi dengan sistem pengolahan data menggunakan PHP, di mana setiap input dari pengunjung melalui "Form Identitas" akan langsung tersimpan dan ditampilkan secara real-time dari database MySQL.
* **Interaksi Modal yang Elegan**: Fitur "Pop-up Lucu" untuk kontak sosial media yang menggunakan efek *glassmorphism* dan *backdrop blur* agar tetap terlihat modern.

## 💻 Detail Teknis

* **Logic (JS)**: Menggunakan ES6+ untuk manipulasi DOM dan logika percabangan waktu.
* **Storage (MySQL)**: Struktur database dirancang untuk menampung data nama, email, alamat, hingga stempel waktu (*timestamp*) pengiriman secara otomatis.
* **Backend (PHP)**: Implementasi `mysqli_real_escape_string` untuk keamanan data serta penggunaan `SESSION` untuk memberikan umpan balik (alert) yang interaktif kepada pengguna.
* **Styling**: Menggunakan Google Fonts "Poppins" untuk tipografi yang tipis dan modern serta teknik *clip-path* untuk memberikan bentuk unik pada header blog.
